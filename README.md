## 📌 Overview

Katefi was born from a simple realization:  
centralized systems can fail — and when they do, entire ecosystems collapse overnight.

Katefi rebuilds the **yap-to-earn economy** on:
- ⚡ Decentralized rails  
- 🔐 Trustless mechanisms  
- 🌍 Permissionless participation  

No gatekeepers. No single points of failure.

## ✨ Features

- 🧠 **AI-powered engagement tracking**

- 🔗 **On-chain transparency & verification**

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/katefi.git
cd katefi
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run locally
```bash
npm run dev
```
